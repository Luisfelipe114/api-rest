instalando o es-lint:
npx eslint --init

instalando nodemon e sucrase:
npm i nodemon sucrase --save-dev

instalando express:
npm i express

instalando dotenv:
npm i dotenv

instalando sequelize e mariadb:
npm i sequelize mariadb

npm i -D(-save--dev) sequelize-cli

instalando bcryptjs:
npm i bcryptjs

instalando o jsonwebtoken
npm i jsonwebtoken


em relação ao sequelize:
*para criar uma migration:
npx sequelize migration:create --name=nome_da_tabela

*após configurada a migration, fazer:
npx sequelize db:migrate

*se quiser desfazer a última migration:
npx sequelize db:migrate:undo
